1️⃣ var, let, const difference: var is function-scoped and can be redeclared, let is block-scoped and can be updated but not redeclared, and const is block-scoped and cannot be reassigned.


2️⃣ Spread operator (...): It expands elements of an array or object into individual elements.


3️⃣ map(), filter(), forEach(): map() returns a new array after transforming items, filter() returns a new array with matched items, and forEach() only loops without returning a new array.


4️⃣ Arrow function: A shorter ES6 syntax for writing functions using =>.

5️⃣ Template literals: ES6 string syntax using backticks `` that allows embedding variables with ${}.
